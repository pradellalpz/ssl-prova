# Exercício 1 – Preparação do Ambiente

## O que é o OpenSSL?
OpenSSL é uma biblioteca de código aberto que implementa os protocolos SSL e TLS.
Ela permite criar certificados digitais, chaves criptográficas e conexões seguras.

## Comandos executados
- sudo apt install openssl -y → instala o OpenSSL
- openssl version → exibe a versão instalada
