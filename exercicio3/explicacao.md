# Exercício 3 – Geração da Chave Privada

## O que é uma Chave Privada?
A chave privada é um arquivo secreto usado para criptografar e assinar
dados. Ela nunca deve ser compartilhada com ninguém.

## Comando executado
- openssl genrsa -out aluno.key 2048
  → gera uma chave privada RSA de 2048 bits e salva no arquivo aluno.key
