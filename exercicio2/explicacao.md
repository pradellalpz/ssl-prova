# Exercício 2 – Estrutura de Diretórios

## O que foi feito
Criação do diretório /etc/ssl/certificates para armazenar certificados digitais.

## Comandos executados
- mkdir -p /etc/ssl/certificates → cria o diretório (e subdiretórios se necessário)
- cd /etc/ssl/certificates → entra no diretório criado

## Por que esse diretório?
O /etc/ssl/ é o local padrão no Linux para armazenar
certificados e chaves SSL. Organizar os arquivos aqui
segue as boas práticas de segurança do sistema.
